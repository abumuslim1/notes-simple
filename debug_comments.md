# Диагностика проблемы с комментариями

## Наблюдения
1. Комментарий добавляется в БД (видно после обновления страницы)
2. Счетчик комментариев обновляется (было 2, стало 3)
3. НО: новый комментарий не появляется в списке без обновления страницы
4. Ошибок в консоли браузера нет

## Возможные причины
1. invalidate() не срабатывает корректно
2. Проблема с кэшированием tRPC
3. Оптимистичное обновление не работает

## Решение
Использовать оптимистичное обновление вместо invalidate для мгновенного отображения комментария
